# Маркетинг Air Sign

Набор материалов для позиционирования, поиска и внешних площадок. Структура согласована с фреймворком [**Marketing Skills**](https://github.com/coreyhaines31/marketingskills) (Corey Haines): базовый контекст лежит в [`.agents/product-marketing-context.md`](../.agents/product-marketing-context.md) — его должны читать агенты и люди перед задачами по копирайту, SEO и запускам.

## Файлы

| Документ | Назначение |
|----------|------------|
| [SEO-KEYWORDS-AND-INTENTS.md](./SEO-KEYWORDS-AND-INTENTS.md) | Запросы, интенты, FAQ, идеи под контент и сниппеты |
| [MESSAGING-SOCIAL-AND-ADS.md](./MESSAGING-SOCIAL-AND-ADS.md) | Тезисы, био, объявления, email одной строки |
| [CONTENT-AND-PAGES.md](./CONTENT-AND-PAGES.md) | Столпы контента, страницы «альтернатив», программатик-SEO |
| [GITHUB-AND-OFFSITE.md](./GITHUB-AND-OFFSITE.md) | Оформление репозитория, личный бренд, каталоги |

## Как ставить skills marketingskills в проект (опционально)

```bash
npx skills add coreyhaines31/marketingskills --skill product-marketing-context copywriting page-cro ai-seo schema-markup
```

После установки агент подтянет `.agents/product-marketing-context.md` как источник правды о продукте.
